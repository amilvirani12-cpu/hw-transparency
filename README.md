# Hewlett-Woodmere UFSD — Independent Fiscal Dashboard

An independent, resident-maintained site that turns the Hewlett-Woodmere school district's
public financial documents into readable dashboards. Not affiliated with the district.

**Live site:** (enable GitHub Pages: Settings → Pages → deploy from `main` branch, root)

## Sections

| Section | Source documents | Generated by |
|---|---|---|
| `budget/` | Monthly WinCap Revenue Status + Budget Status PDFs | `Budget/budget_visualizer.py` |
| `treasurer/` | Monthly Treasurer's Reports | `HW-Treasurer-Report` parser |
| `facilities/tours/` | Board facilities tour documents | `HW-Facilities[Tour]/facility-dashboard` |
| `facilities/bcs/` | NY State Building Condition Survey | `Budget/bcs_visualizer` |

All source documents are public records from the district's
[BoardDocs site](https://go.boarddocs.com/ny/hwps/Board.nsf/Public).

## Updating

Dashboards are currently generated locally by the tools above and copied in
(`budget/FY2026-<Month>.html`, section `index.html` files). A BoardDocs watcher that
automates this via GitHub Actions is planned — see project notes.

## Data policy

This repository contains only public financial records and derived analysis.
No voter data, no personal information.
